# HomeDashboard Add-ons for Home Assistant

This repository contains add-ons for Home Assistant that enhance the HomeDashboard iOS app experience.

## Add-ons

### HomeDashboard Relay

Connect your HomeDashboard app to your local Home Assistant from anywhere - no port forwarding or Nabu Casa required.

[![Open your Home Assistant instance and show the add add-on repository dialog with a specific repository URL pre-filled.](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2FAmpelTech%2Fhomedashboard-ha-addon)

## Installation

1. Click the button above, or manually add this repository URL in Home Assistant:
   ```
   https://github.com/AmpelTech/homedashboard-ha-addon
   ```

2. Install the "HomeDashboard Relay" add-on

3. Get a pairing code from the HomeDashboard app:
   - Open **Settings** > **Remote Access** > **HomeDashboard Relay** > **Set Up**

4. Enter the pairing code in the add-on configuration

5. Start the add-on
